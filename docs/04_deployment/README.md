## 部署架构

### 环境分类
- **开发环境**: 本地开发测试
- **预发布环境**: 功能验证和性能测试
- **生产环境**: 线上服务

### 核心服务
- **Streamlit前端**: 用户界面服务
- **FastAPI后端**: API服务（端口8124）
- **PostgreSQL**: 业务数据库
- **Redis**: 缓存服务

## 运维要点

### 监控指标
- API响应时间
- 数据库连接状态
- 缓存命中率
- 系统资源使用率

### 关键配置
- 数据库连接池配置
- LLM API密钥管理
- 日志级别设置
- 缓存策略配置

## 故障排查

### 常见问题
1. **数据库连接问题** - 检查环境变量配置
2. **API超时问题** - 检查TuShare API状态
3. **缓存问题** - 检查Redis连接
4. **LLM调用失败** - 检查API密钥和配额

### 排查工具
- Docker logs
- 系统日志
- 数据库查询日志
- API监控面板
